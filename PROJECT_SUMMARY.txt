🚀 QUICK START GUIDE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. CREATE VIRTUAL ENVIRONMENT:
   python3 -m venv venv
   source venv/bin/activate

2. INSTALL DEPENDENCIES:
   pip install -r requirements.txt
   python -m spacy download en_core_web_sm

3. CONFIGURE API KEYS:
   cp .env.example .env
   # Edit .env and add your OPENAI_API_KEY

4. RUN APPLICATION:
   streamlit run app.py

5. OPEN IN BROWSER:
   http://localhost:8501

1. API KEYS REQUIRED:
   - Get OpenAI key: https://platform.openai.com/api-keys
   - Get Gemini key: https://makersuite.google.com/app/apikey
   - Add to .env file (DO NOT commit to git)

2. spaCy MODEL:
   - Run: python -m spacy download en_core_web_sm
   - Required for NLP text processing

3. FAISS SETUP:
   - For GPU support, replace 'faiss-cpu' with 'faiss-gpu' in requirements.txt
   - Current setup uses CPU (faiss-cpu) for compatibility

4. FILE SIZE LIMITS:
   - Recommended PDF size: < 10 MB
   - Max PDF size (config): 50 MB
   - Larger files can be split before upload
